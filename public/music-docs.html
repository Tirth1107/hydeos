Core Features Explained

1. Audio Normalization

Hyde Music uses the Web Audio API to analyze the decibel level of incoming streams in real-time. It automatically gains or reduces volume to ensure a consistent listening experience, so you don't have to constantly adjust your volume knob between songs.

2. Smart Caching (Data Saver)

When you listen to a song, Hyde saves it to your device's IndexedDB. If you play that song again, it loads instantly from your disk instead of using your internet bandwidth. This works on both the Web Player and Windows App.

3. Privacy Architecture

Unlike Spotify or YouTube, we do not require an account.
- Playlists are stored locally in a JSON file on your device.
- Search queries are routed through an anonymous proxy to prevent tracking.

Installation Guide

Windows Installation

  1. Download the latest HydeMusic-Setup.exe from the download page.
  2. Run the installer.
  3. Since we are an independent developer and do not purchase expensive certificates, Windows may show a "SmartScreen" popup.
  4. Click "More Info" -> "Run Anyway".
  5. The app will install and launch instantly.

Web Player (PWA)

You can install the website as an app on Chrome/Edge:

  1. Visit music.hydeos.qzz.io.
  2. Click the install icon in the address bar (Monitor with down arrow).
  3. Click "Install". It will now appear in your Start Menu / Desktop.

Troubleshooting

Music stops playing in background (Mobile Web)

Fix: Browser battery optimization often kills background tabs. Use the "Install App" (PWA) feature or wait for the native Android app release.

Song fails to load

Fix: This usually happens if the source (YouTube/SoundCloud) has geo-blocked the content. Try skipping to the next song or searching for a generic version (e.g., "Song Name Lyrics") instead of the official video.

Windows App: "Unsafe File" Warning

Reason: This is expected for indie software. We do not buy corporate signing certificates from Microsoft. The app is virus-free and stable; simply click "Run Anyway".
